# PART 1: PROMPT ENGINEERING FUNDAMENTALS

## How AI Actually Works (Non-Technical Version)

AI language models are like extremely well-read assistants who've consumed billions of pages of text. They predict what words should come next based on patterns they've learned.

**Key insight:** AI doesn't "know" things—it recognizes patterns. Your job is to give it enough context to find the right patterns.

### What This Means for Your Prompts

**More context = Better output**

Bad: "Write me a caption"
Better: "Write me an Instagram caption for a dentist promoting teeth whitening"
Best: "Write me an Instagram caption for a family dentist in Miami promoting teeth whitening. Tone: warm, professional, friendly. Target: women 35-55 who want a brighter smile for an upcoming event. Include a soft CTA to book a consultation."

---

## The Anatomy of a Perfect Prompt

Every high-performing prompt contains these elements:

### 1. ROLE (Who is the AI?)
Tell the AI what expert to embody.

*Example:* "You are a direct-response copywriter with 20 years of experience writing for dental practices..."

### 2. CONTEXT (What's the situation?)
Provide background information.

*Example:* "My client is a family dentist in a competitive suburban market. They want to attract more cosmetic dentistry patients..."

### 3. TASK (What specifically should it do?)
Be explicit about the deliverable.

*Example:* "Write 5 Instagram captions promoting teeth whitening services..."

### 4. FORMAT (How should the output look?)
Specify structure, length, and style.

*Example:* "Each caption should be 150-200 words, include emojis sparingly, end with a question to drive comments..."

### 5. CONSTRAINTS (What to avoid?)
Set boundaries to prevent bad outputs.

*Example:* "Avoid medical jargon, don't make unrealistic promises, don't use ALL CAPS..."

### 6. EXAMPLES (What does good look like?)
Show the AI what you want.

*Example:* "Here's a caption style I like: [paste example]..."

---

## The RICE Framework for Better Outputs

Use this checklist before running any prompt:

**R - Role:** Did you tell the AI who to be?
**I - Information:** Did you provide enough context?
**C - Command:** Is the task crystal clear?
**E - Examples:** Did you show what good looks like?

If output is bad, usually one of these is missing.

---

## Common Mistakes and How to Fix Them

### Mistake 1: Being Too Vague
**Bad:** "Write content for my business"
**Fix:** Add specifics about your business, audience, and goals

### Mistake 2: Not Specifying Tone
**Bad:** "Write a LinkedIn post about productivity"
**Fix:** "Write a LinkedIn post about productivity. Tone: professional but conversational, like talking to a colleague over coffee"

### Mistake 3: Accepting First Output
**Bad:** Using whatever the AI generates first
**Fix:** Use refinement prompts (see below) to improve outputs

### Mistake 4: Overloading One Prompt
**Bad:** Asking for 10 things at once
**Fix:** Break complex requests into steps

### Mistake 5: No Examples
**Bad:** Describing what you want abstractly
**Fix:** Paste examples of content you like and say "write something similar"

**Limitations:**

**Limitations:**
- Writing quality varies

Start with ChatGPT or Claude for most content creation. Use the same prompts from this guide—they work across all major AI models.

## Full Script Prompts by Format

### Short-Form Script (15-60 seconds)

```
You are a viral short-form video scriptwriter specializing in [INDUSTRY] content.

Write a [15/30/60]-second video script about [TOPIC].

Context:
- Target audience: [DESCRIBE]
- Pain point addressed: [PAIN POINT]
- Goal of video: [AWARENESS/TRUST/CONVERSION]
- Brand voice: [DESCRIBE TONE]

Structure the script as:
1. HOOK (first 3 seconds): Stop the scroll
2. PROBLEM (5-10 seconds): Identify the pain
3. AGITATION (5-10 seconds): Make it feel urgent
4. SOLUTION (10-20 seconds): Present your answer
5. CTA (5 seconds): Tell them what to do

Format the output as:
- Spoken words (what to say)
- Visual cues [in brackets]
- Timing markers

Constraints:
- No jargon unless audience knows it
- Every sentence must move the story forward
- CTA must be specific and actionable
```

### Long-Form Script (2-10 minutes)

```
You are an educational content creator who makes complex topics simple and engaging.

Write a [LENGTH]-minute video script about [TOPIC].

Context:
- Platform: [YOUTUBE/COURSE/WEBINAR]
- Target audience: [DESCRIBE]
- Their current knowledge level: [BEGINNER/INTERMEDIATE/ADVANCED]
- What they should be able to DO after watching: [OUTCOME]

Structure:
1. HOOK (0:00-0:30): Pattern interrupt + promise
2. CONTEXT (0:30-1:30): Why this matters to them
3. MAIN CONTENT (1:30-[X]):
   - Point 1: [CONCEPT] with example
   - Point 2: [CONCEPT] with example
   - Point 3: [CONCEPT] with example
4. SUMMARY (last 1 minute): Key takeaways
5. CTA (last 30 seconds): What to do next

Include:
- Transitions between sections
- Engagement questions to keep viewers watching
- Specific examples from [INDUSTRY]
- Common mistakes to avoid

Avoid:
- Lengthy introductions
- Tangents that don't serve the main point
- Vague advice without concrete steps
```

### Testimonial/Case Study Script

```
You are a storytelling expert who turns client results into compelling narratives.

Write a [LENGTH] testimonial-style video script.

Client info:
- Industry: [CLIENT'S INDUSTRY]
- Starting point: [BEFORE STATE]
- End result: [AFTER STATE]
- Timeline: [HOW LONG]
- Key actions taken: [WHAT THEY DID]

Structure as a transformation story:
1. HOOK: Lead with the result or transformation
2. THE "BEFORE": Paint a vivid picture of the struggle
3. THE DISCOVERY: How they found the solution
4. THE JOURNEY: Key moments and breakthroughs
5. THE "AFTER": Specific, tangible results
6. THE LESSON: What others can learn
7. CTA: How viewers can get similar results

Make it feel:
- Authentic (not salesy)
- Relatable (viewers should see themselves)
- Specific (real numbers and details)
- Emotional (connect to feelings, not just facts)
```

## Refinement Prompts for Scripts

Use these after getting initial output:

**Make it punchier:**
```
Rewrite this script to be 30% shorter while keeping all key points. Cut filler words and get to the point faster.
```

**Increase energy:**
```
Rewrite this script with higher energy. Add more dynamic language, shorter sentences, and stronger verbs.
```

**Add specificity:**
```
Rewrite this script adding specific numbers, names, and examples. Replace vague claims with concrete details.
```

**Simplify:**
```
Rewrite this script at an 8th-grade reading level. Replace jargon with simple words. Use shorter sentences.
```

**Add hooks throughout:**
```
Add pattern interrupts and open loops throughout this script so viewers stay engaged. Insert a micro-hook every 30 seconds.
```
