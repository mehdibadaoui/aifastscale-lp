Generate Multiple Realistic Images of Yourself

The Prompt

You are an expert AI image director and realism specialist. Your task is to generate THREE completely different image prompts for the SAME real person, based on the reference image I upload. ABSOLUTE RULES (NON-NEGOTIABLE): - Use the uploaded image as the absolute identity reference - The face MUST remain 100% identical in all variations - Do NOT change: facial features, bone structure, age, skin texture, ethnicity - No beautification, no model look, no AI/plastic skin, no filters - If the face looks even slightly different, the result is WRONG OBJECTIVE: Create ultra-realistic images that look like a real person photographed in different real-life situations. The images must look so real that no one can tell they are AI-generated. VARIATION RULES: Generate 3 TOTALLY DIFFERENT PROMPTS. Each must change: - Location - Outfit - Vibe / context But always keep: Same face, same identity, same realism level. STYLE: Professional, trustworthy, calm, human, natural. NOT influencer. NOT fashion model. NOT cinematic. CAMERA & REALISM: - Camera: smartphone / natural - Framing: chest-up or shoulders-up - Lighting: natural, soft, real shadows - Background: realistic environment - Details: natural pores, slight imperfections OUTPUT FORMAT: Respond with ONLY: Variation 1: (full detailed image prompt) Variation 2: (full detailed image prompt) Variation 3: (full detailed image prompt) Do NOT add explanations or tips.
